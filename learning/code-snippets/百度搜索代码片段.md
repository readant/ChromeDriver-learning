# 百度搜索代码片段

## 📝 代码描述

**功能说明：** 实现百度搜索的自动化功能，包括打开百度、输入关键词、点击搜索、获取结果

**使用场景：** 网页搜索自动化、数据抓取、自动化测试

**相关概念：** [[ChromeDriver 基础概念]], [[元素定位方法]]

## 💻 代码实现

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.chrome.service import Service
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

def baidu_search(keyword, max_results=10):
    """
    百度搜索自动化函数
    
    Args:
        keyword (str): 搜索关键词
        max_results (int): 最大结果数量，默认10条
        
    Returns:
        list: 搜索结果列表，每个元素包含标题和链接
    """
    driver = webdriver.Chrome(service=Service("path/to/chromedriver"))
    wait = WebDriverWait(driver, 10)
    
    try:
        driver.get("https://www.baidu.com")
        
        search_box = wait.until(
            EC.presence_of_element_located((By.ID, "kw"))
        )
        search_box.send_keys(keyword)
        
        search_button = wait.until(
            EC.element_to_be_clickable((By.ID, "su"))
        )
        search_button.click()
        
        results = wait.until(
            EC.presence_of_all_elements_located((By.CLASS_NAME, "result"))
        )
        
        search_results = []
        for i, result in enumerate(results[:max_results], 1):
            try:
                title = result.find_element(By.TAG_NAME, "h3").text
                link = result.find_element(By.TAG_NAME, "a").get_attribute("href")
                search_results.append({
                    "index": i,
                    "title": title,
                    "link": link
                })
            except:
                continue
        
        return search_results
        
    finally:
        driver.quit()

if __name__ == "__main__":
    results = baidu_search("ChromeDriver 教程")
    for result in results:
        print(f"{result['index']}. {result['title']}")
        print(f"   {result['link']}\n")
```

## 📖 代码说明

**关键部分：**
1. **显式等待**：使用 WebDriverWait 等待元素加载完成
2. **元素定位**：通过 ID 定位搜索框和搜索按钮
3. **结果提取**：遍历搜索结果并提取标题和链接

**注意事项：**
- 确保 ChromeDriver 路径正确
- 使用显式等待而非 time.sleep
- 添加异常处理提高代码健壮性
- 使用国内网站避免网络问题
