# 百度搜索自动化实践

## 🎯 实践目标

**目标描述：** 使用 ChromeDriver 自动化实现百度搜索功能，包括输入关键词、点击搜索按钮、获取搜索结果

**预期成果：**
1. 成功打开百度首页
2. 自动输入搜索关键词
3. 点击搜索按钮
4. 获取并显示搜索结果

## 📋 实践步骤

### 步骤1：准备工作

**相关概念：** [[ChromeDriver 基础概念]], [[元素定位方法]]

**准备工作：**
1. 安装 ChromeDriver（版本与 Chrome 匹配）
2. 安装 Selenium 库
3. 配置 ChromeDriver 路径

### 步骤2：实施过程

**代码实现：** [[百度搜索代码片段]]

**关键代码：**
```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.chrome.service import Service

# 创建浏览器实例
driver = webdriver.Chrome(service=Service("path/to/chromedriver"))

try:
    # 访问百度首页
    driver.get("https://www.baidu.com")
    
    # 定位搜索框并输入关键词
    search_box = driver.find_element(By.ID, "kw")
    search_box.send_keys("ChromeDriver 教程")
    
    # 定位搜索按钮并点击
    search_button = driver.find_element(By.ID, "su")
    search_button.click()
    
    # 获取搜索结果
    results = driver.find_elements(By.CLASS_NAME, "result")
    print(f"搜索结果数量: {len(results)}")
    
finally:
    driver.quit()
```

**遇到的问题：**
1. 元素定位失败 → [[元素定位失败问题]]
2. 页面加载超时 → [[页面加载超时问题]]

## 💡 经验总结

**成功经验：**
1. 使用 ID 定位元素最稳定
2. 添加 try-finally 确保浏览器关闭
3. 使用国内网站避免网络问题

**失败教训：**
1. 初期未处理页面加载超时导致脚本失败
2. 元素定位方式选择不当导致定位失败

**改进建议：**
1. 添加显式等待机制
2. 增加错误处理和日志记录
3. 封装为可复用的函数

## 🔧 技术细节

**使用的技术：**
- Selenium WebDriver

- 元素定位（ID、Class Name）

**技术难点：**
1. 元素定位的准确性和稳定性

**解决方案：**
1. 使用多种定位方式作为备选
2. 添加等待机制
