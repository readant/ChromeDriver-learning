# 学习系统使用指南

## 📚 系统概述

本学习系统基于 Obsidian 构建，提供完整的个人学习记录和管理功能，支持：

- 📝 学习笔记记录
- 💻 代码片段管理
- 🔗 双向链接和知识图谱
- 🏷️ 标签分类管理
- 📸 图片和视频资源集成
- 📅 每日学习记录
- 🎯 实践项目管理

## 🚀 快速开始

### 1. 安装 Obsidian

1. 下载 [Obsidian](https://obsidian.md/)
2. 安装并打开 Obsidian
3. 选择"打开文件夹"
4. 选择 `learning` 文件夹

### 2. 配置 Obsidian

系统已预配置 Obsidian 设置，包括：
- ✅ 属性管理
- ✅ 标签面板
- ✅ 知识图谱
- ✅ 反向链接
- ✅ 代码高亮

### 3. 开始学习

1. 使用模板创建笔记
2. 记录学习内容
3. 添加双向链接
4. 管理资源文件

## 📝 笔记类型

### 每日记录 (Daily Notes)

**用途：** 记录每日学习进度和计划

**创建方式：**
- 使用快捷键 `Ctrl + D`
- 或使用模板 `templates/daily-note-template.md`

**内容包含：**
- 今日学习计划
- 学习内容记录
- 实践记录
- 思考与总结
- 相关链接

### 概念笔记 (Concept Notes)

**用途：** 记录学习到的概念和知识点

**创建方式：**
- 使用模板 `templates/concept-template.md`

**内容包含：**
- 概念定义
- 核心要点
- 相关概念
- 应用场景
- 示例代码

### 实践笔记 (Practice Notes)

**用途：** 记录实践项目和实验

**创建方式：**
- 使用模板 `templates/practice-template.md`

**内容包含：**
- 实践目标
- 实践步骤
- 实践结果
- 经验总结
- 后续计划

### 问题笔记 (Problem Notes)

**用途：** 记录遇到的问题和解决方案

**创建方式：**
- 使用模板 `templates/problem-template.md`

**内容包含：**
- 问题描述
- 问题分析
- 解决方案
- 问题总结
- 经验教训

### 代码片段 (Code Snippets)

**用途：** 记录可复用的代码片段

**创建方式：**
- 使用模板 `templates/code-snippet-template.md`

**内容包含：**
- 代码描述
- 代码实现
- 代码说明
- 测试用例

## 🔗 双向链接

### 创建链接

```markdown
# 链接到其他笔记
[[ChromeDriver 基础概念]]

# 带文本的链接
[[ChromeDriver 基础概念|ChromeDriver]]

# 链接到特定段落
[[ChromeDriver 基础概念#核心要点]]

# 链接到代码片段
[[code-snippets/百度搜索代码片段]]
```

### 查看反向链接

- 在笔记右侧面板查看"反向链接"
- 点击链接查看引用此笔记的所有笔记

### 知识图谱

- 使用快捷键 `Ctrl + G` 打开知识图谱
- 查看笔记之间的关系网络
- 点击节点跳转到对应笔记

## 🏷️ 标签管理

### 创建标签

```markdown
# 单个标签
#concept

# 多个标签
#concept #ChromeDriver #基础

# 嵌套标签
#concept/ChromeDriver/基础
```

### 查看标签

- 在左侧面板查看"标签"面板
- 点击标签查看所有相关笔记

### 标签规范

**推荐标签：**
- `#concept` - 概念笔记
- `#practice` - 实践笔记
- `#problem` - 问题笔记
- `#code` - 代码片段
- `#daily` - 每日记录

**主题标签：**
- `#ChromeDriver`
- `#Selenium`
- `#元素定位`
- `#等待策略`

## 📸 资源管理

### 添加图片

```markdown
# 相对路径引用
![图片描述](../resources/images/screenshot.png)

# Obsidian 风格引用
![[resources/images/screenshot.png]]
```

### 添加视频

```markdown
# 链接到视频文件
[视频演示](../resources/videos/demo.mp4)
```

### 资源命名规范

- 图片：`{日期}-{主题}-{序号}.png`
- 视频：`{日期}-{主题}-{类型}.mp4`

详见 [[资源管理指南]]

## 🎯 最佳实践

### 1. 笔记命名

- 使用清晰、描述性的标题
- 使用中文标题便于搜索
- 避免特殊字符

### 2. 内容组织

- 使用标题层级组织内容
- 使用列表列举要点
- 使用代码块展示代码

### 3. 链接使用

- 及时添加相关链接
- 使用双向链接建立知识网络
- 定期检查和更新链接

### 4. 标签管理

- 使用一致的标签体系
- 避免创建过多标签
- 定期整理和合并标签

### 5. 资源管理

- 按规范命名资源文件
- 及时清理无用资源
- 定期备份重要资源

## 🔧 高级功能

### 1. 模板使用

- 使用快捷键 `Ctrl + E` 插入模板
- 自定义模板内容
- 模板变量：`{{date}}`, `{{time}}`

### 2. 属性管理

每个笔记都有属性：
- `created`: 创建日期
- `modified`: 修改日期
- `status`: 状态
- `category`: 分类
- `tags`: 标签

### 3. 搜索功能

- 使用快捷键 `Ctrl + Shift + F` 全局搜索
- 使用 `Ctrl + F` 当前笔记搜索
- 支持正则表达式搜索

### 4. 导出功能

- 导出为 Markdown
- 导出为 PDF
- 导出为 HTML

## 🆘 常见问题

### Q: 如何备份笔记？

A: 定期复制 `learning` 文件夹到备份位置，或使用云存储同步。

### Q: 如何分享笔记？

A: 可以导出为 Markdown 或 PDF 格式分享给他人。

### Q: 如何迁移到其他设备？

A: 复制整个 `learning` 文件夹到新设备，用 Obsidian 打开即可。

### Q: 如何恢复误删的笔记？

A: 使用 Git 版本控制恢复，或从备份中恢复。
