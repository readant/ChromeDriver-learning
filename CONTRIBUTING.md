# Contributing to ChromeDriver Learning

欢迎贡献代码！请遵循以下指南。

## 代码规范

- 使用 Python 3.8+
- 遵循 PEP 8 代码风格
- 使用 `black` 格式化代码
- 添加必要的注释

## 提交规范

```
feat: 添加新功能
fix: 修复 bug
docs: 更新文档
refactor: 重构代码
test: 添加测试
chore: 日常维护
```

## 贡献流程

1. Fork 仓库
2. 创建分支 (`git checkout -b feature/xxx`)
3. 提交更改 (`git commit -m 'feat: xxx'`)
4. 推送到分支 (`git push origin feature/xxx`)
5. 创建 Pull Request

## 联系方式

如有问题，欢迎提交 Issue。
